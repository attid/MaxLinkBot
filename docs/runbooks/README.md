# Runbooks

Эта директория зарезервирована под эксплуатационные инструкции.

Минимальный набор runbooks, который должен появиться вместе с кодом:
- первичный запуск в Docker;
- восстановление после `reauth_required`;
- диагностика отсутствия новых сообщений из MAX;
- диагностика ошибки создания Telegram topics;
- проверка целостности SQLite и topic mappings.
